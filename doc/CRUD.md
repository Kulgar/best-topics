# CRUD

**"Verbes" HTTP:**
GET / POST / PATCH | PUT / DELETE

* **HTTP        <-> CRUD              <-> Rails**
*   POST        <-> CREATE            <-> create
*   GET         <-> READ              <-> show
*   PATCH / PUT <-> UPDATE            <-> update
*   DELETE      <-> DELETE / DESTROY  <-> destroy

**On retrouve aussi en Rails :**
* index
* new
* edit
